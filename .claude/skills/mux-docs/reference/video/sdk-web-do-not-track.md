# _guides/snippets/sdk-web-do-not-track

**Source:** https://mux.com/docs/_guides/snippets/sdk-web-do-not-track

By default, Mux plugins for HTML5-based players do not respect Do Not Track when set within browsers. This can be enabled in the options passed to Mux, via a setting named respectDoNotTrack. The default for this is false. If you would like to change this behavior, pass respectDoNotTrack: true.
